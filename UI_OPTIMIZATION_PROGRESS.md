### 2. 合并选择工具面板 ✓

**完成时间**: 2026-06-27  
**文件**: [ToolPanel.java](src/main/java/com/beatblock/ui/panels/ToolPanel.java)

#### 改进内容

**之前**:
- 选择工具在 ToolPanel
- 选择属性在 SelectionPropertiesPanel（需要手动打开）
- 两个面板分离，用户困惑

**之后**:
- **集成设计**：工具属性直接显示在工具下方
- **动态显示**：根据选中的工具自动显示相应参数
- **分组清晰**：工具设置 + 通用设置

#### 集成的功能

**工具特定属性**:
- 笔刷：形状（球体/立方体）+ 大小（1-32）
- 线选：线粗细（0-32）
- 魔棒/连接：扩散半径（1-256）+ 完整状态匹配
- 平面切片：切片朝向（7个方向）
- 其他工具：显示"（无特殊设置）"

**通用属性**:
- 操作模式：新建/加选/减选（单选按钮）
- 相机距离：16-512（滑块）
- 包含空气：复选框
- 选区统计：实时显示已选方块数

#### UI布局

```
┌─────────────────────────────────────┐
│ 方块选择工具                         │
│ [工具下拉框：笔刷]                   │
│                                     │
│ 工具设置:                            │
│ ├ 形状: [球体 ▼]                    │
│ └ 大小: ●────────── 16              │
│                                     │
│ 通用设置:                            │
│ ├ 操作: ◉新建 ○加选 ○减选           │
│ ├ 相机距离: ●────── 128             │
│ ├ ☑ 包含空气                        │
│ └ ✓ 已选: 24 个方块                 │
└─────────────────────────────────────┘
```

#### 效果

- ✅ 面板切换次数：2次 → **0次**
- ✅ 参数可见性：隐藏 → **立即可见**
- ✅ 操作流畅度：⭐⭐⭐☆☆ → **⭐⭐⭐⭐⭐**
- ✅ SelectionPropertiesPanel：必需 → **可选**（高级用户仍可使用）

## 💡 技术亮点

### 1. 自动命名算法
```java
private String generateAutoObjectName() {
    int counter = 1;
    while (objectExists("selection_" + counter)) {
        counter++;
    }
    return "selection_" + counter;
}
```
