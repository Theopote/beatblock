# BeatBlock

BeatBlock 是一个音乐可视化创作工具：音频分析的作用是辅助创作者快速找到节拍点、生成初稿，但最终呈现的每一个方块事件都是时间轴上的、可编辑、可重放的确定性数据，不依赖运行时重新分析音频。

## 中心模型（三层单向流）

```
第 1 层 音频参考轨（只读，可手改）  →  BPM / 节拍点 / 段落 / 能量曲线
第 2 层 时间轴事件（权威编辑层）    →  StageEvent + CameraEvent
第 3 层 播放器（只消费第 2 层）      →  BlockAnimationEngine + 相机系统
```

三种典型效果（建筑出现、跑酷敲击、镜头跟随下落）是同一抽象的不同参数组合：**舞台对象 + 时刻动作 + 相机运动**。

详见 **[架构文档](docs/architecture.md)**（概念类型与代码类映射、三种效果参数表）。

## 产品定位

BeatBlock 面向**被观看的演出**（录屏、存档回放），不是玩家实时按键的节奏游戏。时间轴是权威数据源；音频播放时钟只对齐预览进度，不是事件来源。
